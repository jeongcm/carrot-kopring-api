# carrot-kopring-backend-server
## kotlin 기반 spring 프로젝트

---
## 환경
### jdk 17, gradle 8.3, spring boot

## 사용한 스택
### mariadb, spring security, jpa, queryDsl, oauth2, amazon S3, oidc(진행중)

---
- 빌드
```bash
./gradlew clean build
```

- 로컬 Run
```bash
./gradlew bootRun
```

- Docker build
```bash
docker build -t imageName:tag .
```
